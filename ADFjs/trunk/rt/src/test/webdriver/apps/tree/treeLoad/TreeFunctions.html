Steps
1. Make sure Suppliers and References nodes are expanded initially.
2. Click on the button 'Add context menu to collapsible tree'
3. Tree should launch as a context menu on right-click or Shift-F10
4. Click Edit -> cut / copy /paste / remove. All these use standard menu handler and should work.
5. Click on the button 'Remove context menu to collapsible tree'. This should remove the context tree.
